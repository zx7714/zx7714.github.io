随便写点

1.官网

  • 中文官网：http://www.redis.cn
  • 英文官网：https://redis.io

ps:很重要，包含redis从低阶到高阶所有文当，包括一些知识点的解释说明

2.安装过程（Ubuntu）

  1. 创建一个你要安装的目录(不会可以退群了)，我放在了/opt/software下，执行wget,地址可以去官网找
    # wget http://download.redis.io/releases/redis-5.0.7.tar.gz
  2. 解压,进入到解压出的文件
    # tar -xvzf redis-5.0.7.tar.gz
  3. 查看README，所有步骤都有写
    # vim.tiny README.md
  4. 完！！！
